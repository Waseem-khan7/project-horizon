# Project Horizon 🚀

Project Horizon is a frontend application built with **React**, **TypeScript**, and **Vite**. The project follows modern development practices with a focus on scalability, maintainability, and responsive user interface development.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Redux Toolkit
- React Router DOM

## Getting Started

### 1. Clone the repository

```
git clone https://github.com/Waseem-khan7/project-horizon.git
```

### 2. Navigate to the project directory

```
cd project-horizon
```

### 3. Install dependencies

```
npm install
```

### 4. Start the development server

```
npm run dev
```
